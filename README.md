# Android Projects

This repository contains a collection of Android and Flutter applications developed during the Mobile Application Development Training Program conducted by my college. These projects demonstrate my learning journey, covering Android development with Java, Flutter development, Firebase integration, UI design, API integration, and mobile application concepts.

## Technologies Used

* Java
* Android Studio
* Flutter
* Firebase Authentication
* Firebase Realtime Database
* REST APIs
* XML
* Material Design

## Projects Included

* **Audify** – Music player application
* **Captain** – Voice automation app
* **CareerSuggestion** – Career guidance application
* **Day8** – Training practice project
* **DietPlanner** – Diet planning application
* **FitLife** – Fitness application
* **FitMyLife** – Health & fitness application
